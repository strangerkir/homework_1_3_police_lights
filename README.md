# Police Lights

A simple PlatformIO project for creating a police light effect with LEDs.

## Overview

This repository contains firmware for a microcontroller that runs a police-style flashing light pattern.

## Build

Use PlatformIO to build and upload the firmware to the target board.

## Usage

Connect the LEDs to the required GPIO pins, then upload the project using PlatformIO.

## Notes

- Adjust pin assignments and timing in the source code as needed.
- Ensure the target board is supported by PlatformIO.
